Frequently Asked Questions

Find quick answers about Aigle — usage, backups, and technical notes.

Aigle is a lightweight inventory and analytics system for managing medicines, transactions, and deliveries. It uses a Flask backend with Jinja templates and a simple client-side JS layer for interactive tables and charts.

  • Backend: Flask (Python)
  • Templates: Jinja2 (`template/`)
  • Static assets: `static/` (CSS, JS, images)
  • Styles: `static/css/style.css` + page styles like `analytics.css`
  1. Login with your account.
  2. Use the sidebar to navigate between Dashboard, Inventory, and Transactions.
  3. Admins can access Employees and Analytics pages.
  4. Use the search box in the top-right to highlight rows in tables.

To backup your data, download the database file here: /link

Alternatively, export via the admin panel (if available) or run your database dump tools directly against the database file.

Routes are defined in blueprints located in the `routes/` directory. Static files are served from `static/`. If you need to change the port, edit `app.py`.
